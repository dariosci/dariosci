## 🌱 Proyecto destacado

### AgroTrace

Sistema de trazabilidad para empaques frutícolas:

* Recepción de fruta por lote
* Control de calidad
* Producción y packing
* Gestión de operarios y turnos
* Generación de pallets
* Despachos con packing list
* Trazabilidad completa mediante QR
* Dashboard de rendimiento
